# Transformice Fishing Event

# Relic locations

Users can add a relic missing from a location.

This is saved locally but also sent to a server.

If mutiple reports are made for a relic/location, it is considered _verified_ and will be displayed to everyone.

# What happens when relic locations change?

The position of relics was reset twice in the 2026 event.

The server will interpret a burst of submissions for new relic locations as a reset.

Clients will not receive older submissions and will delete older local data.

Old submissions are preserved so a false positive can be rolled back by deleting this reset date on the server.
